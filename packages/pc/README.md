# train-admin-frontend

## 目录结构

```

src
├── api          // 接口
├── assets       // 资源文件
├── components   // 组件
├── constant     // 全局通用定义
├── img          // 图片
├── interface    // ts 相关定义
├── views        // 页面
└── store        // store

```

## 开发相关

1. 安装 git

2. 安装 node

3. 开发前安装依赖
  pnpm i

4. 启动服务
  pnpm start
